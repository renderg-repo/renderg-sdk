renderg_utils
================

renderg_utils 是 RenderG SDK 的工具包，提供了各种通用的工具函数和类，支持 SDK 其他部分的正常运行。

该包包含以下主要功能：

- 系统路径管理和工作空间配置
- JSON 文件的读写操作
- 日志记录和日志管理
- 常量定义和异常处理
- 命令行执行和进程管理
- 通用工具函数和辅助方法
