renderg_api
==============

renderg_api 是 RenderG SDK 的核心 API 包，提供了与 RenderG 渲染服务进行交互的完整功能集。

该包包含以下主要功能：

- 与 RenderG 服务建立安全连接
- 管理用户账户和权限
- 创建、提交和管理渲染作业
- 监控和控制渲染任务
- 管理项目和环境配置
- 处理文件传输任务
- 提供消息队列连接功能
